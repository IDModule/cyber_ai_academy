# أكاديمية سايبر الذكية 🛡️

منصة تعليمية تفاعلية مبتكرة في مجال **الأمن السيبراني**، مدعومة بالذكاء الاصطناعي، مصممة خصيصًا للفئة العمرية من 12 إلى 21 سنة.

---

## 🎯 عن المنصة

أكاديمية سايبر هي منصة تعلم إلكتروني تهدف إلى تعليم الشباب أساسيات الأمن السيبراني بطريقة تفاعلية وممتعة، مبنية على أحدث نظريات التعلم والتصميم التعليمي.

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 📚 ثلاثة أبواب تعليمية | المفاهيم الأساسية، الحماية الرقمية، التهديدات المتقدمة |
| 📝 6 دروس تفاعلية | درسان في كل باب مع محتوى غني ومتدرج |
| 🧪 تمارين عملية | أسئلة متعددة الخيارات، صح/خطأ، وسيناريوهات واقعية |
| 📊 اختبارات تقييمية | اختبار في نهاية كل باب مع تصحيح فوري |
| 🤖 مساعد ذكاء اصطناعي | مساعد "سايبر" يتكيف مع عمر المتعلم |
| 🏆 لوحة صدارة | تنافس بين المتعلمين بناءً على النقاط والإنجازات |
| 🎖️ شارات تحفيزية | نظام مكافآت لتعزيز الدافعية |
| 📜 شهادات إتمام | شهادات مخصصة قابلة للتنزيل |
| 📈 تتبع التقدم | لوحة تحكم شخصية مع إحصائيات مفصلة |

## 🏗️ البنية التقنية

- **الواجهة الأمامية:** React 19 + Tailwind CSS 4
- **الخادم:** Express + tRPC
- **قاعدة البيانات:** MySQL (Drizzle ORM)
- **الذكاء الاصطناعي:** LLM مدمج للتغذية الراجعة والمساعدة
- **التصميم:** متجاوب بالكامل، داعم للغة العربية (RTL)

---

## 🚀 التشغيل

```bash
# تثبيت الاعتماديات
pnpm install

# تشغيل بيئة التطوير
pnpm dev

# بناء المشروع للإنتاج
pnpm build
pnpm start
```

## 📋 متطلبات التشغيل

- Node.js 18+
- قاعدة بيانات MySQL
- متغيرات البيئة (DATABASE_URL, JWT_SECRET, إلخ)

---

## 🎨 التصميم

المنصة مصممة بأسلوب أنيق يعكس هوية الأمن السيبراني:
- ألوان داكنة مع تدرجات سيان وأخضر
- أيقونات وعناصر بصرية تعكس عالم الحماية الرقمية
- تصميم متجاوب يعمل على جميع الأجهزة
- دعم كامل للغة العربية واتجاه RTL
